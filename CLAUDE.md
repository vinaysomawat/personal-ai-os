## Project Overview

Vinay AI OS is a personal AI Operating System designed to support decision-making, automate repetitive thinking, and track key life areas. It is organized into domain modules:

- **Planner** — scheduling, tasks, priorities
- **Career** — job tracking, goals, progress
- **Finance** — budgeting, expenses, investments
- **Health** — fitness, habits, wellness
- **Learning** — courses, notes, skill tracking
- **Coding** — projects, snippets, references
- **Documents** — files, summaries, knowledge base

## Status
